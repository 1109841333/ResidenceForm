## ResidenceForm 领地Form支持
为领地添加基岩版Form菜单以让基岩版玩家可以更好的使用领地
## 已实现的功能
* 领地选择
* 领地公共权限管理 ( res set )
* 领地玩家管理 ( res pset )
* 领地信任玩家管理 ( res padd/pdel )
* 领地扩展/缩小 ( res expand/contract )
## 待实现的功能
* 领地传送 ( res tp )
* 领地删除 ( res remove )
* 领地转移 ( res give )
* 领地踢出 ( res kick )
* 领地重命名 ( res rename )
* 领地创建选择 ( res select )
* 领地创建草稿 ( 独有功能 )
* 领地创建 ( res create )
* 领地边界查看 ( res show )
* 领地信息 ( res info )
## 前置
* Floodgate (2.0版本以上) - 用于基岩版菜单
* Residence (5.0版本以上) - 领地插件
